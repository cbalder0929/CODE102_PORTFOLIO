03Image Detect

This project is a browser-based image classifier built with HTML, CSS, JavaScript, and TensorFlow.js using the pre-trained MobileNet model loaded from CDN scripts. Users can drag and drop or upload an image, which is validated, previewed, and analyzed directly in the browser with no backend required. The JavaScript handles model loading, progress updates, file handling, and calls `model.classify()` to generate predictions and confidence scores. The interface includes a responsive card layout, animated progress bar, and interactive drag-and-drop upload area for a smooth user experience.

04API Weather App

WeatherData is a frontend-only weather app built with HTML, CSS, and JavaScript that uses OpenStreetMap Nominatim for geocoding and the National Weather Service API for forecast data. The app converts user-selected locations into coordinates, fetches weather grid data, and dynamically updates current conditions, forecast cards, and Chart.js visualizations directly in the browser with no backend required. All API handling, data transformation, error handling, and UI rendering are managed in script.js, creating a fast lightweight architecture focused on real-time weather visualization.
